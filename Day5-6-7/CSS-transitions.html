The novel coronavirus scientifically termed with the name SARS-Cov-2 or COVID-19 has infected millions and millions of people worldwide. Creator: Naeblys , Credit: Getty Images/iStockphoto Many web and mobile applications are available to track the progress of the disease. The dashboards show the live updates of the daily cases, recoveries, deaths, active cases, the number of tests conducted, and so on. Most of the dashboards provide interactiveness only for the maps and not for other kinds of visualizations. The main necessity to develop this application is to give more control to the user so that he can provide his input to visualize varieties of interactive charts and results. The application also allows the user even to view the very minute data points available in the chart by zooming, panning, and tooltip. The web application provides the answer to the users and also allows them to download every chart created.
